## API-method `saveconfig`

### Signature
``` c++
bool saveconfig(struct[] Options);
```

### Description
Saves configuration file to the disk.

### Arguments
- **Options** `(struct[])`
  - **Name** `(string)` - Option name.
  - **Value** `(string)` - Option value.

### Return value
`true` on success or `false` on failure.
