## API-method `testextension`

## Since 
`v23.0`

### Signature
``` c++
bool testextension(string ExtEntryName);
```

### Description
Tries to find an executor program, e.g. Pytohn for the extension.

### Arguments
- `ExtEntryName` - Extension entry filename.

### Return value
`true` on success or `false` on failure.
